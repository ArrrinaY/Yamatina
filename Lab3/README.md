## Основные интерфейсы коллекций
- IEnumerable<T> - базовый интерфейс для поддержки перебора элементов (foreach)
- ICollection<T> - добавляет методы для управления коллекцией (Add, Remove, Clear, Count)
- IList<T> - расширяет ICollection поддержкой индексации и операций по индексу
- IDictionary<TKey, TValue> - определяет контракт для коллекций пар "ключ-значение"

## Реализованные структуры данных

### 1. SimpleList<T> - Динамический массив
Реализуемые интерфейсы: IEnumerable<T>, ICollection<T>, IList<T>

### 2. SimpleDictionary<TKey, TValue> - Хэш-таблица с цепочками
Реализуемые интерфейсы: IDictionary<TKey, TValue>, IReadOnlyDictionary<TKey, TValue>

### 3. LinkedList<T> - Двусвязный список
Реализуемые интерфейсы: IEnumerable<T>, ICollection<T>, IList<T>
