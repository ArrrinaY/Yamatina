# Классы и структура проекта:
Модели для задачи "Обедающие философы" (Dining Philosophers):
1. DinningPhilosopherWithDeadlock.cs - реализация с гарантированным deadlock (демонстрация проблемы)

2. DinningPhilosphersWithoutDeadlock.cs - исправленная версия без deadlock

Модель для задачи "Спящий парикмахер" (Sleeping Barber):
1. BarberShop.cs - реализация парикмахерской с ограниченной очередью клиентов

Модели для задачи "Производитель-Потребитель" (Producer-Consumer):
1. ProducerConsumerBlockingCollection.cs - реализация с использованием BlockingCollection<T>

2. ProducerConsumerSemaphore.cs - реализация с использованием SemaphoreSlim + lock
